# grpc-example
This is an example gRPC server definition that follows Youtuber [Dream of Code's](https://www.youtube.com/@dreamsofcode) tutorial video, [Tonic makes gRPC in Rust Stupidly Simple](https://youtu.be/kerKXChDmsE?si=lGkxKwF7dOIXUmKw). I have added personal comments for better understanding.

### Misc Comments

- Tonic, Tokio, and Prost are usable as long as Protoc is installed. You don't necessarily need to pull/reference the repo
